打开[https://python.org/](url)下载最新版的Python，打开Python IDLE，在左上角的`File`中点击`New File`，并输入以下这一串Code，用来导入Turtle
```Python
import turtle
```
指令介绍：`forward()`使画笔前进
`right()`使画笔右转

配合使用
```Python
import turtle
turtle.forward(100)
turtle.right(90)
turtle.forward(100)
turtle.right(90)
turtle.forward(100)
turtle.right(90)
turtle.forward(100)
turtle.right(90)
```
即可画出一个正方形
